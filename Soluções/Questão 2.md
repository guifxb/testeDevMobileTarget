# Resolução do Desafio 2 - Target Sistemas

## Descrição do Problema

Dada uma sequência de números, precisamos identificar a lógica que gera essa sequência e completar com o próximo elemento. Abaixo está a explicação para cada sequência:

### a) 1, 3, 5, 7, ___

- **Lógica**: Esta é uma sequência de números ímpares consecutivos.
- **Próximo número**: 9.

### b) 2, 4, 8, 16, 32, 64, ____

- **Lógica**: Cada número é o dobro do anterior. Ou seja, a lógica é uma multiplicação por 2.
- **Próximo número**: 128.

### c) 0, 1, 4, 9, 16, 25, 36, ____

- **Lógica**: Esta é uma sequência de quadrados perfeitos (números ao quadrado). 0², 1², 2², 3², 4², 5², 6², e assim por diante.
- **Próximo número**: 49 (7²).

### d) 4, 16, 36, 64, ____

- **Lógica**: Esta sequência segue uma Progressão Aritmética (PA), onde a diferença entre os termos aumenta em 8 unidades a cada passo.
- **Próximo número**: 100.

### e) 1, 1, 2, 3, 5, 8, ____

- **Lógica**: Esta é a famosa sequência de Fibonacci, onde cada número é a soma dos dois anteriores.
- **Próximo número**: 13 (5 + 8).

### f) 2, 10, 12, 16, 17, 18, 19, ____

- **Lógica**: A lógica por trás desta sequência é que todos os números começam com a letra "D" (dois, dez, doze, etc.)
- **Próximo número**: 200.
